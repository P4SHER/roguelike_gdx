## 🎯 Что изучено

### ✅ Domain Layer (Игровая логика)
- **entities/** - Player, Enemy, Character, Stats, Item, Backpack (12 файлов)
- **level/** - Level, Room, Tile, LevelGeneration, Coordinates (10 файлов)
- **service/** - GameService, CombatService, MovementService, EnemyAiService, InventoryService (14 файлов)

### ✅ Data Layer (Сохранение)
- **SaveService.java** - главный сохранитель
- **JsonMapper.java** - Jackson сериализация
- **11 DTO файлов** - для всех сущностей
- JSON файлы: savegame.json, leaderboard.json

### ✅ Presentation Layer (Рендеринг)
- **LibGDX слой** - 6-слойный рендеринг
  - TileLayerRenderer, ActorLayerRenderer, ItemLayerRenderer
  - FogLayerRenderer, EffectsLayerRenderer, UILayerRenderer
- **Экраны** - MenuScreen, GameScreen, InventoryScreen, LeaderboardScreen
- **Входа** - InputHandler, InputQueue
- **Lanterna слой** - для тестирования

### ✅ Функционал игры
- 21 уровень с процедурной генерацией
- 9 комнат (3x3) на каждом уровне
- 5 типов врагов с разным ИИ
- 5 типов предметов с бонусами
- Пошаговая боевая система
- Полное сохранение/загрузка
- Таблица рекордов
