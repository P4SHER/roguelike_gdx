## 🆘 Частые вопросы

### "Где главная логика игры?"
**Ответ:** `GameService.java` - это сердце игры

### "Где враги?"
**Ответ:** `domain/entities/Enemy.java` + `domain/level/LevelGeneration.java` (где спауны)

### "Где сохранение?"
**Ответ:** `datalayer/SaveService.java` + `datalayer/data/savegame.json`

### "Как добавить нового врага?"
**Ответ:** QUICK_REFERENCE.md → таблица "Я хочу изменить" → "Типы врагов"

### "Как изменить урон?"
**Ответ:** PROJECT_STRUCTURE.md → "CombatService" → `calculateDamage()`

### "Где рендеринг?"
**Ответ:** `lwjgl3/src/main/java/io/github/example/presentation/renderer/layers/`
