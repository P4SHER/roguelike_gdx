USAGE QUICK START
=============================================================================

1. Basic transition (0.5 second default crossfade):
   presentationLayer.transitionToScreen(newScreen);

2. Custom duration (0.75 seconds):
   presentationLayer.transitionToScreen(newScreen, 0.75f);

3. Custom transition type:
   ScreenTransition fadeOut = new FadeOutTransition(0.3f);
   screenManager.transitionToScreen(newScreen, fadeOut);

4. State preservation:
   GameStateSnapshot snapshot = new GameStateSnapshot(gameSession, "GameScreen");
   screenManager.saveGameState("GameScreen", snapshot);
